<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    What is Super Hexagon?
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-12">
    Super Hexagon is a fast-paced, reaction-based arcade game where you control a small triangle rotating around a central hexagon. Your goal is to avoid incoming obstacles by rotating left or right while the hexagon continuously spins. This project recreates the intense gameplay experience using modern web technologies.
  </p>

  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4 mt-16">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Super Hexagon in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Super Hexagon repository to your local machine:

      ```bash
      git clone https://github.com/miguelean/super-hexagon.git
      cd super-hexagon
      ```
    </Step>

    <Step title="Install dependencies">
      Install the required packages using your preferred package manager:

      <CodeGroup>
      ```bash npm
      npm install
      ```

      ```bash yarn
      yarn install
      ```

      ```bash pnpm
      pnpm install
      ```
      </CodeGroup>
    </Step>

    <Step title="Start the development server">
      Run the Next.js development server:

      ```bash
      npm run dev
      ```

      Open [http://localhost:3000](http://localhost:3000) in your browser to play the game.
    </Step>
  </Steps>
</div>
